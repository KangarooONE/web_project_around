# 📸 Proyecto Sprint 7: Around the U.S.
Este proyecto es una galeria interactiva de fotografías con diseño responsivo

## 🔎 Funcionalidades Implementadas
- Apertura de popup para edición de perfil mediante botón
- Cierre del popup con botón dedicado
- Edición de nombre de usuario
- Edición de descripción del perfil
- Actualización inmediata del perfil en la interfaz
- Precarga de datos actuales del perfil al abrir el popup
- Validación de formulario:
  - El botón de guardar se deshabilita si los campos están vacíos
  - El botón se habilita únicamente cuando ambos campos tienen contenido
- Sin recarga de página al enviar el formulario

## 🤖 Tecnologías utilizadas

- HTML5
- CSS3 (estructura modular tipo BEM)
- JavaScript (Vanilla JS)

## Estructura del Proyecto

📁 Proyecto
│
├── 📁 blocks/
│   └── (componentes CSS - BEM)
│
├── 📁 pages/
│   └── index.css
│
├── 📁 scripts/
│   └── index.js
│
├── 📁 vendor/
│   ├── normalize.css
│   └── fonts.css
│
├── 📁 images/
│   └── (assets del proyecto: fotos, iconos)
│
├── 📄 index.html
└── 📄 README.md

- **blocks/** → componentes CSS (metodología BEM)
- **pages/** → archivo principal de estilos (index.css)
- **scripts/** → lógica JavaScript del proyecto
- **vendor/** → librerías externas (normalize, fonts)
- **images/** → recursos gráficos (íconos, fotos, UI assets)
